# 2026-07-18

## 新增 CLI 命令行入口
- 新建 `src/cli.py`，支持以下命令：
  - `info` — 展示当前全部信息（版本、机器码、端口、积分缓存、API Key、上游 Key、设置）
  - `credits` — 查询积分（调服务端 + 显示本地缓存）
  - `redeem <卡密>` — 卡密充值
  - `start` — 开启代理服务（自动获取 BuddyKey，显示 API Key，Ctrl+C 停止）
  - `config` — 展示客户端配置 JSON
- `main.py` 检测 sys.argv[1] 是否为 CLI 子命令，是则走 CLI 不启动 GUI
- 用法：`python -m src.cli info` 或 `python src/main.py info`

## setup_debug_port.py 纯 ctypes COM 重写
- 快捷方式操作全部用 `ctypes.windll.ole32` 直接调 COM（CoCreateInstance + vtable），不依赖 pywin32
- 关闭进程用 `CreateToolhelp32Snapshot` + `TerminateProcess`
- 启动用 `os.startfile` 或 `ShellExecuteW`，不弹命令窗口
- 从 requirements.txt 移除 pywin32 依赖
