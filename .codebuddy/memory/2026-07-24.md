## 修复 macOS Dock 不显示问题（src/main.py）

### 问题
macOS 上 Nuitka onefile 生成的是纯可执行文件（非 `.app` bundle），默认不会出现在 Dock 和应用切换器 (Cmd+Tab) 中，用户关闭窗口后无法找到应用。

### 修复
在 `QApplication` 创建后添加 macOS Dock 显示逻辑：
1. 尝试用 `AppKit.NSApp.setActivationPolicy_(1)` 设置为普通应用（Regular）
2. `AppKit` 不可用时回退到纯 ctypes 方式调用 `libobjc.dylib`
3. `setActivationPolicy_(1)` = NSApplicationActivationPolicyRegular，让程序出现在 Dock
4. `activateIgnoringOtherApps_(True)` 强制激活窗口
